Ecommerce Website

With Guest User Cart and Paypal Intregration

The project is an Ecommerce website where user can view and buy products or added them to cart to save the products for future purchases.

This website has a feature where a user can buy a product even if they have not been registered as a customer

For the Payment gateway, Paypal has been used so users can pay for the products via Paypal or using their Debit/Credit card

There are mainly two types of products available i.e Digital and non-digital products, where both have their own method of payment associated with them.

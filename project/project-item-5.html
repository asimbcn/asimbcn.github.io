Auction Website

With Automated Bidding and Assigning System

The project is an Auction system where user can view and bid for products or added them to their wishlist to view or bid for the products in the future.

This website has a feature where a user can bid for a product and if all the necessary criteria is met, is assigned the top bidder status

The products have an expiration date, which when reached, the product is automatically set as closed and the product is automatically added to the cart of the customer who is the highest bidder in that instance of the related product.

There is also a custom made admin panel where the admin can control various aspects of the user experience.

Users can also edit their profile easily and edit their shipping address once after they first register. If they want to edit their shipping address, admin can provide a OTA for the user.

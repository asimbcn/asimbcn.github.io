Portfolio Website

Dynamic Website Using Django

This is a Dynamic Portfolio Web Application. It support multi-user access with selective data visualizer in the frontend. It is fully featured consisting a custom admin panel for users to signup and create a porfolio just by adding few information.

The front-end is fully dynamic and the contents of the currently active user is displayed at a time.

Multiple Users can create their portfolio at the same time but only the user who is active is referenced for the Frontend to display data.

Users can also upload their CV/Resume, which can be downloaded from the frontend.
